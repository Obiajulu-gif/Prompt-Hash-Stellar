# PromptHash Stellar Overview

## Summary

PromptHash Stellar is an early-stage Soroban application for selling encrypted AI prompt licenses with XLM payments and wallet-verified access. The project is aimed at creators who want a repeatable revenue model for digital knowledge products and at buyers who want a verifiable way to prove access rights on-chain.

## Product Vision

The long-term vision is to make Stellar a credible settlement and rights-management layer for digital commerce, starting with prompt assets. PromptHash Stellar treats prompts as licensed digital goods rather than speculative tokens. That framing is important because the core business action is pay-to-access, not transfer-to-own.

## Target Users

- AI prompt creators monetizing reusable workflows
- agencies and consultants packaging internal playbooks as paid assets
- operators, marketers, researchers, and founders buying specialist prompt packs
- Stellar developers looking for a practical reference app built around Soroban commerce flows

## Primary Use Cases

- creator lists a premium prompt with a free preview
- buyer pays in XLM to unlock the full asset
- platform records entitlements in contract state and uses wallet proof for delivery
- creator updates pricing or pauses a listing without migrating to a new token model

## Why It Matters For Stellar

PromptHash Stellar adds a useful application category to the ecosystem:

- it increases XLM utility in creator-economy commerce
- it shows how Soroban can manage rights and fee routing for non-financial products
- it creates a reusable pattern for applications that need both payment and controlled access

## Current Status

The repository is functional as a foundation and honest about its stage. The contract, frontend flows, and unlock endpoints are already present. The project still needs more indexing, testing, and deployment hardening before it should be treated as production-ready.
